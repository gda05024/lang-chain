### Setup Requirements

  - Python 3.8 이상 설치
  - OpenAI API 키
  - terminal


### Get started

1. LangChain 관련 라이브러리 설치
```
pip install langchain langchain-openai langchain-community
pip install langchain-text-splitters langchain-postgres
```
2. OpenAI API 키 설정
```
export OPENAI_API_KEY=sk-xxxxxx
```
